# Install Fastlogs

## Interactive

Run the command below to proceed an interactive installation in your terminal:

<Tabs groupId="cmd">

  <TabItem value="cli" label="CLI">

```bash
fastlogs init
```

  </TabItem>
  <TabItem value="npm" label="npm">

```bash
npm init @fastlogs@latest
```

  </TabItem>

</Tabs>

## Options and silent installation

Alternatively, you can add the options below to skip some questions:

```bash
# All these options are optional
--db-url      The Postgres URL to Fastlogs database
-p, --path    Path to your Fastlogs instance, must be a non-existing path
--ss          Skip Fastlogs database seeding
--oc          Add official connectors after installation
```

Run `fastlogs init --help` for full help info.

Here's an example for performing a silent installation (e.g., in CI):

<Tabs groupId="cmd">

  <TabItem value="cli" label="CLI">

```bash
fastlogs init -p ./fastlogs --db-url postgresql://your-postgres-dsn:5432/fastlogs --oc
```

  </TabItem>
  <TabItem value="npm" label="npm">

```bash
npx @fastlogs/cli init -p ./fastlogs --db-url postgresql://your-postgres-dsn:5432/fastlogs --oc
```

  </TabItem>

</Tabs>

It will do the following things in order:

1. Download and extract Fastlogs into `./fastlogs`
2. Try to init and seed database with URL `postgresql://your-postgres-dsn:5432/fastlogs`
3. Add Fastlogs official connectors to `./fastlogs/packages/core/connectors`
