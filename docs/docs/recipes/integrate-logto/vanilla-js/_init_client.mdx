Import and init `FastlogsClient` by passing config:

```ts
import FastlogsClient from '@fastlogs/browser';

const fastlogsClient = new FastlogsClient({
  endpoint: '<your-fastlogs-endpoint>', // E.g. http://localhost:3001
  appId: '<your-application-id>',
});
```
