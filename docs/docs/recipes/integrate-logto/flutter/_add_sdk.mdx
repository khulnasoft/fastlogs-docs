Our flutter SDK package is published on [pub.dev](https://pub.dev/packages/fastlogs_dart_sdk).

```bash
flutter pub get fastlogs_dart_sdk
```

Or you may directly get your own copy of the SDK from [github](https://github.com/fastlogs-docs.khulnasoft.com/dart)

```bash
git clone https://github.com/fastlogs-docs.khulnasoft.com/dart
```
