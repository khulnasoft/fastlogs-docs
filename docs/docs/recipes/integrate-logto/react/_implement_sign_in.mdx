We provide two hooks `useHandleSignInCallback()` and `useFastlogs()` which can help you easily manage the authentication flow.

<SignInNote calling=".signIn()" />

Go back to your IDE/editor, use the following code to implement the sign-in button:

```tsx
import { useFastlogs } from '@fastlogs/react';

const SignIn = () => {
  const { signIn, isAuthenticated } = useFastlogs();

  if (isAuthenticated) {
    return <div>Signed in</div>;
  }

  return <button onClick={() => signIn('http://localhost:3000/callback')}>Sign In</button>;
};
```
