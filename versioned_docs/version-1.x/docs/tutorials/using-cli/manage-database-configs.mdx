# Manage database configs

Fastlogs helps to maintain some technical configs, such as OIDC and Cookie keys. Before we can managing them via Admin Console or Management API, you can use CLI to achieve the goal.

## Get config by key

Usage:

<Tabs groupId="cmd">

  <TabItem value="cli" label="CLI">

```bash
fastlogs db config get <key> [keys...]
```

  </TabItem>
  <TabItem value="npx" label="npx">

```bash
npx @fastlogs/cli db config get <key> [keys...]
```

  </TabItem>

</Tabs>

Say you want to get the OIDC private keys:

<Tabs groupId="cmd">

  <TabItem value="cli" label="CLI">

```bash
fastlogs db config get oidc.privateKeys
```

  </TabItem>
  <TabItem value="npx" label="npx">

```bash
npx @fastlogs/cli db config get oidc.privateKeys
```

  </TabItem>

</Tabs>

The output will be like:

```bash
oidc.privateKeys=["\n-----BEGIN PRIVATE KEY-----\nMIIJRAIBA..."]
```

Run `fastlogs db config get --help` for all available keys, or see [Configuration](/docs/references/core/configuration#database-configs) for detailed explanation.

## Set config by key

<Tabs groupId="cmd">

  <TabItem value="cli" label="CLI">

```bash
fastlogs db config set <key> [keys...]
```

  </TabItem>
  <TabItem value="npx" label="npx">

```bash
npx @fastlogs/cli db config set <key> [keys...]
```

  </TabItem>

</Tabs>

Say you want to set the OIDC Cookie keys:

<Tabs groupId="cmd">

  <TabItem value="cli" label="CLI">

```bash
fastlogs db config set oidc.cookieKeys "[\"key1\",\"key2\"]"
```

  </TabItem>
  <TabItem value="npx" label="npx">

```bash
npx @fastlogs/cli db config set oidc.cookieKeys "[\"key1\",\"key2\"]"
```

  </TabItem>

</Tabs>

Normally, the value to set should be a valid JSON string, and follows the pre-defined type definition. Run `fastlogs db config set --help` for all available keys, or see [Configuration](/docs/references/core/configuration#database-configs) for detailed explanation.
