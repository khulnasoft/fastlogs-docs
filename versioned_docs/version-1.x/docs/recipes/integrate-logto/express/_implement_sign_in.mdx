### Prepare fastlogs routes

Prepare routes to connect with Fastlogs.

Go back to your IDE/editor, use the following code to implement the API routes first:

```ts
import { handleAuthRoutes } from '@fastlogs/express';

app.use(handleAuthRoutes(config));
```

This will create 3 routes automatically:

1. `/fastlogs/sign-in`: Sign in with Fastlogs.
2. `/fastlogs/sign-in-callback`: Handle sign-in callback.
3. `/fastlogs/sign-out`: Sign out from Fastlogs.

### Implement sign-in

We're almost there! Now, create a sign-in button to redirect to the sign-in route on user click.

```ts
app.get('/', (req, res) => {
  res.setHeader('content-type', 'text/html');
  res.end(`<div><a href="/fastlogs/sign-in">Sign In</a></div>`);
});
```
