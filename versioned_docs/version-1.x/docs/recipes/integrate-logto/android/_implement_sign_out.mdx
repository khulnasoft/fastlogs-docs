Calling `.signOut(completion)` will always clear local credentials even if errors occurred.

<Tabs>

<TabItem value="kotlin" label="Kotlin">

```kotlin
fastlogsClient.signOut { fastlogsException: FastlogsException? ->
    // Local credentials are cleared regardless of whether `fastlogsException` is null.
}
```

</TabItem>

<TabItem value="java" label="Java">

```java
fastlogsClient.signOut(fastlogsException -> {
    // Local credentials are cleared regardless of whether `fastlogsException` is null.
});
```

</TabItem>

</Tabs>
